## Installation

Install GPT-3 with npm

```bash
  # Clone this repository
  $ git clone https://github.com/dannyberns/react-gpt3-landingpage

  # Go into the repository
  $ cd react-gpt3-landingpage

  # Install dependencies
  $ npm install

  # Run the app
  $ npm start
```
